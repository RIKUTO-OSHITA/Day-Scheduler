# Day Schedule App

Outlook 風の 1 日スケジュール管理 UI を HTML/CSS/JavaScript で実装したサンプルです。

## 使い方

1. [index.html](index.html) をブラウザで開きます。
2. 左上に開いている日付が表示されます。
3. 日付入力で表示日を変更できます。
4. タイムラインをクリックすると開始時刻が自動入力されます。
5. 開始時刻と終了時刻は 07:30-18:00 のプルダウンから選択できます。
6. タスク入力フォームでタイトル・開始時刻・終了時刻を入力して保存します。

## 仕様

- 時間範囲: 07:30-18:00
- 開始/終了時刻: 15 分刻み（入力時に自動正規化）
- 開始/終了時刻入力: 07:30-18:00 のプルダウン選択
- タイムライン時間軸ラベル: 30 分刻み
- タイムライン表示: 横並び 2 列の一覧（縦スクロール不要）
- 出力形式: ブラウザで直接開ける HTML
